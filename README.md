RAY is a resume-driven job opportunity discovery platform.
Upload a resume and receive relevant, verified job opportunities.

## Features

- Resume-based job matching
- Real job opportunities only
- Match scoring
- Job freshness indicators
- Skill demand insights
- Save & export jobs

## Tech Stack

- Python
- Streamlit
- NLP (spaCy, TF-IDF)
- Real job APIs (Adzuna, JSearch, Workday)

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
